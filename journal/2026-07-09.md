## Decisions (and why)
- **UI = Ontop's built-in YASGUI (`localhost:7300`), not a custom app.** It already ships syntax
  highlighting + results + right-click "View SQL translation"; no HTTP API exposes the SQL
  translation (a custom UI would need an `ontop` CLI shell-out). Fewer moving parts. (Memory:
  `ui-decision-builtin-yasgui`.)
- **Password never in a mounted file.** `postgres.properties` holds only `jdbc.url` + driver; the
  credential is injected via `ONTOP_DB_PASSWORD` from `secrets/.env`. Satisfies CLAUDE.md #2 + the
  secrets directive with no render step.
- **`ground_truth` over `oracle`/`base`.** "Oracle" reads as the DB vendor here; "base" is vague.
  Port `7300` dodges 8080 (Airflow/Tomcat/Trino) and pairs with GraphDB's 7200.
